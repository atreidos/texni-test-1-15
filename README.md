# Трекер задач V1

Простой прототип трекера задач на чистом HTML/CSS/JS.

## Структура

- `index.html` — разметка страницы.
- `styles.css` — базовые стили.
- `app.js` — логика добавления, отметки и удаления задач.

## Запуск

1. Склонировать репозиторий:
   ```bash
   git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>.git
   cd <ИМЯ_ПАПКИ_РЕПОЗИТОРИЯ>
   ```
2. Открыть файл `index.html` в браузере (двойной клик или через контекстное меню "Open With").

Никаких зависимостей и сборки не требуется.
